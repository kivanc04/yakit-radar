Yakıt Radar

Yakıt Radar is a Flutter-based mobile application that helps drivers quickly find the most suitable fuel stations nearby. The app analyzes distance, station availability, and fuel quality indicators to guide users toward the best refueling option.

Instead of simply listing stations, Yakıt Radar prioritizes the most relevant choices and provides fast navigation access so drivers can reach their destination efficiently.

⸻

Key Features

Nearby Station Detection

When the application starts, it automatically detects and displays the closest fuel station to the user. The distance to the station is shown in meters so users can quickly evaluate proximity.

Station Availability

Each station displays whether it is open or closed, allowing users to avoid navigating to unavailable stations.

One-Tap Navigation

Users can tap the navigation button to instantly open map navigation and receive directions to the selected fuel station.

Smart Recommendation

Besides the nearest station, the application can suggest the most optimal station, considering additional factors such as overall station conditions and fuel quality indicators.

Fuel Price Alerts

The app provides notifications about expected fuel price changes. For example, users can see alerts such as upcoming fuel price drops so they can decide whether to refuel immediately or wait.

Saved Locations

Users can save frequently used locations such as home, work, or custom destinations, making it easier to find nearby fuel stations relative to those locations.

Interactive Map

All fuel stations are visible on an interactive map, allowing users to explore available stations and select them directly.
